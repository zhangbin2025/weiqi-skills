🔧 技能一：SGF棋谱打谱网页生成器

功能概述

将 SGF 棋谱文件转换为单文件 HTML 交互式打谱网页，支持标准SGF格式和野狐围棋格式（含变化图）。

Python JavaScript 单文件输出

核心文件位置

  • 转换脚本：/root/.openclaw/workspace/qipu/sgf_to_html.py
  • 网页模板：/root/.openclaw/workspace/qipu/viewer.html
  • 下载棋谱：/root/.openclaw/workspace/qipu/YYYY-MM-DD/*.sgf

技术实现原理

1. SGF解析逻辑

Python - 主分支提取算法
# 核心逻辑：depth连续递增，每个depth只有一个着法
depth_groups = {}  # 按深度分组
for move in moves:
    d = move['depth']
    if d not in depth_groups:
        depth_groups[d] = []
    depth_groups[d].append(move)

# 提取主分支（忽略变化图）
main_moves = []
prev_depth = -1
for d in sorted(depth_groups.keys()):
    group = depth_groups[d]
    if d == prev_depth + 1:
        main_moves.append(group[0])  # 取第一个着法
        if len(group) > 1:  # 多个着法=变化图开始
            break
    prev_depth = d

2. 格式检测

  • 嵌套格式：野狐原始格式，含深度信息 (depth)
  • 平面格式：已清理的标准SGF，按顺序排列

3. HTML生成

JavaScript - 棋盘渲染核心
// 19x19棋盘绘制

const ctx = canvas.getContext('2d');

🔧 技能三：云比赛网数据查询

功能概述

获取云比赛网(yunbisai.com)的围棋比赛数据，包括比赛列表、分组信息、对阵表、选手成绩等。

REST API JSON 实时数据

数据源

  • 平台：云比赛网 (yunbisai.com)
  • 数据类型：围棋比赛、分组信息、对阵表、选手成绩

API端点列表

1. 获取比赛列表

API - 比赛列表
GET https://data-center.yunbisai.com/api/lswl-events

参数:
- page: 页码 (1)
- eventType: 2 (围棋)
- month: 最近多少个月 (1, 3, 6, 12)
- areaNum: 区域（URL编码，如广东省=%E5%B9%BF%E4%B8%9C%E7%9C%81）
- PageSize: 每页数量 (50)

示例:
https://data-center.yunbisai.com/api/lswl-events?page=1&eventType=2&month=1&areaNum=%E5%B9%BF%E4%B8%9C%E7%9C%81&PageSize=50

2. 获取比赛分组信息

API - 分组信息
GET https://open.yunbisai.com/api/event/feel/list

参数:
- event_id: 比赛ID
- page: 1
- pagesize: 500

示例:
https://open.yunbisai.com/api/event/feel/list?event_id=64665&page=1&pagesize=500

返回字段:
- participantname: 选手姓名
- groupname: 组别名称
- rank_num: 排名
- integral: 积分
- vicsum: 胜场数
- faisum: 负场数

3. 获取对阵详情

API - 对阵表
GET https://api.yunbisai.com//request/Group/Againstplan

参数:
- groupid: 分组ID
- bout: 轮次 (1, 2, 3...)

示例:
https://api.yunbisai.com//request/Group/Againstplan?groupid=317032&bout=1

返回字段:
- total_bout: 总轮数
- p1, p2: 选手姓名
- p1_score, p2_score: 得分（0=负, 1=和, 2=胜）
- p1_result, p2_result: 结果代码

排名计算规则

1. 积分规则

  • 胜：2分
  • 和：1分
  • 负：0分

2. 对手分计算

所有已赛对手的个人积分之和（轮空对手不计入）

3. 累进分计算

累进分示例（4胜1负）
第1轮后：2分
第2轮后：4分  
第3轮后：6分
第4轮后：8分
第5轮后：8分（负）
累进分 = 2+4+6+8+8 = 28分

4. 排名优先级

  1. 个人积分（高者优先）
  2. 对手分（高者优先）
  3. 累进分（高者优先）
