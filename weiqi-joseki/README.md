# weiqi-joseki

围棋定式四角八向变化图生成器

## 功能

输入一个围棋定式，输出该定式在棋盘四个角的8种对称变化图。

## 快速开始

```bash
python3 scripts/joseki_transform.py '(;B[qp];W[oq];B[pq];W[op];B[qn];W[kq])'
```

## 输出示例

生成的8种变化：
1. 原始定式（根据输入自动检测角位）
2. 左上角变化（lurd）
3. 左上角变化（ludr）
4. 左下角变化（ldru）
5. 左下角变化（ldur）
6. 右上角变化（ruld）
7. 右上角变化（rudl）
8. 右下角变化（rdlu）
9. 右下角变化（rdul）

## 坐标系说明

SGF坐标系：以左上角为原点，横向a→t，纵向a→t

8个变化坐标系：以各角为原点，不同轴向组合
