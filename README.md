# 围棋技能包集合 (Weiqi Skills)

> **每一位围棋爱好者，都值得拥有属于自己的围棋智能助手。**

这不是一套大而全的工业化围棋软件。我们相信，每个棋手的棋风、学习路径、关注领域都独一无二——有人痴迷AI定式演变，有人偏爱古谱意境，有人需要比赛备战工具，有人只想记录日常对局。

**weiqi-skills** 是一套模块化、可组合的围棋技能包，让你像搭积木一样构建**个性化的围棋工作流**：

- 📚 收录**属于你自己的棋谱库**，而非被平台算法推荐
- 🎯 从个人对局中**提炼专属定式**，而非死记硬背标准答案
- 🔧 按需组合技能包，打造**符合你棋风的AI助手**

每个技能包都是独立的，你可以自由选择和组合，让技术真正服务于你的围棋之路。

---

## 🤖 开发方式

**weiqi-skills 是由 AI 自动化开发的围棋技能包集合。**

这代表着一种全新的开源项目开发模式：

- **AI 驱动开发** — 所有代码、文档、测试均由 AI 生成和迭代
- **人机协作** — 人类负责提出需求和方向，AI 负责实现和优化
- **快速迭代** — 从想法到可用功能，通常只需几轮对话
- **模块化架构** — 每个技能包独立开发，按需组合

> 💡 **这不仅是围棋工具，也是 AI 辅助开发的探索实验。**

---

## 🔒 隐私与数据政策

**weiqi-skills 承诺：**

- ✅ **不收集个人数据** - 我们不会收集、存储或传输任何用户的个人信息
- ✅ **无内置数据** - 项目本身不包含任何棋谱、定式或其他围棋数据
- ✅ **仅提供开放数据获取能力** - 所有技能包只提供从公开渠道获取数据的工具能力
- ✅ **数据本地存储** - 你的棋谱库、定式库全部保存在你自己的设备上

你的围棋数据，完全属于你自己。

## 典型工作流

### 🔄 闭环：从观赛到复盘

```
[野狐观赛] → [weiqi-foxwq 下载] → [weiqi-db 入库]
    → [weiqi-joseki 提取定式] → [weiqi-sgf 生成打谱页] → [复盘学习]
```

**适用场景**：追完一场精彩的职业对局，想深入研究其中的定式变化和关键手筋。

---

### 📝 闭环：从对局到成长

```
[面棋/网棋对局] → [weiqi-recorder 记谱] → [weiqi-db 保存]
    → [个人棋谱库积累] → [weiqi-joseki 发现自己的惯用定式] → [针对性训练]
```

**适用场景**：记录自己的对局，长期积累后分析个人棋风和常见误算模式。

---

### 🏆 闭环：从备战到赛场

```
[weiqi-yunbisai 查比赛] → [weiqi-player 查对手段位] → [weiqi-fetcher 下载对手棋谱]
    → [weiqi-sgf 研究对手棋风] → [制定开局策略]
```

**适用场景**：比赛前了解对手信息，针对性准备开局。

## 功能详解

### 🗄️ weiqi-db - 围棋棋谱数据库
- 本地棋谱管理工具，统一收纳野狐、本地SGF等来源的棋谱
- 支持SGF导入、元数据编辑、标签管理
- 全文搜索、JSON查询语法，AI友好接口
- 数据存储于 `~/.weiqi-db/database.json` 单文件

### 🔽 weiqi-fetcher - 分享棋谱下载器
从各大围棋平台分享链接自动下载SGF棋谱，支持10个平台：
- OGS、野狐围棋、101围棋网
- 弈客围棋、元萝卜围棋、星阵围棋
- 隐智智能棋盘、弈客少儿版
- 弈城围棋、腾讯围棋
- 自动解析各种链接格式，一键下载SGF文件

### 🔽 weiqi-foxwq - 野狐棋谱下载
- 按日期批量下载野狐围棋棋谱
- 支持从分享链接实时提取棋谱（WebSocket协议解析）
- 支持进行中/已结束对局

### 📚 weiqi-joseki - 围棋定式数据库
- 定式录入与管理，支持8向对称变化自动生成
- 从棋谱批量提取定式，自动检测脱先（PASS）定式
- KataGo Archive 自动导入，支持断点续传和内存保护
- 定式冲突检测与棋谱定式匹配识别
- 数据存储于 `~/.weiqi-joseki/database.json`

### 🔍 weiqi-player - 棋手查询
- 查询手谈等级分（dzqzd.com）
- 查询业余段位（yichafen.com）
- 批量查询支持

### 📝 weiqi-recorder - 记谱工具
- 单文件网页版记谱工具
- 支持黑白轮流落子、提子、打劫规则、悔棋
- 无需安装，浏览器直接打开使用

### 🎮 weiqi-sgf - SGF打谱工具
- 将SGF棋谱转换为交互式HTML网页
- 支持播放/暂停/手数跳转
- 支持让子棋

### 🏆 weiqi-yunbisai - 比赛查询
- 查询云比赛网围棋比赛列表
- 获取分组信息和选手名单
- 计算完整排名（积分/对手分/累进分）
