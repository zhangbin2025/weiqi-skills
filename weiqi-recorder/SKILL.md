# 围棋记谱工具

单文件 HTML 围棋记谱工具，无需服务器，直接在浏览器中运行。

## 功能特点

- **棋盘显示**：标准 19×19 路围棋棋盘
- **轮流落子**：自动切换黑白双方
- **提子处理**：自动检测并移除无气棋子
- **打劫规则**：严格版打劫，标记禁入点，停一手/悔棋解除
- **悔棋功能**：支持撤销上一步，恢复被提棋子
- **手数显示**：实时显示当前手数
- **响应式设计**：支持移动端触摸操作

## 使用方法

### 提供记谱工具给用户

**重要**：QQ 文件传输有缓存机制，直接发送同一文件可能显示旧文件名。发送前复制到 /tmp 并添加时间戳注释（改变文件内容即可绕过缓存）：

```bash
cp weiqi-recorder/assets/weiqi_recorder.html /tmp/weiqi_recorder.html && echo "<!-- $(date +%s) -->" >> /tmp/weiqi_recorder.html
```

然后发送 `/tmp/weiqi_recorder.html`

或直接指导用户在浏览器中打开 `assets/weiqi_recorder.html` 文件。

### 使用方式

1. **落子**：点击棋盘交叉点落子
2. **悔棋**：点击悔棋按钮撤销上一步
3. **清空**：重新开始一盘新棋
4. **打劫规则**：
   - 单提形成打劫时会自动标记禁入点
   - 停一手或悔棋可解除打劫状态

## 技术说明

- **纯前端**：单 HTML 文件，内嵌 CSS 和 JavaScript
- **Canvas 绘制**：使用 HTML5 Canvas 绘制棋盘和棋子
- **无依赖**：不依赖任何外部库
- **离线可用**：保存到本地后无需网络连接
